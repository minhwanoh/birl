# birl
A Python Implementation of Bayesian Inverse Reinforcement Learning (BIRL)

## Prerequisites

- Python 2.7
- SciPy
- pymdptoolbox (https://media.readthedocs.org/pdf/pymdptoolbox/latest/pymdptoolbox.pdf)
- CFFI
- Python Boost
